## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* git
* JDK

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yesmanic/Java_Chess
   ```
2. Compile the project
   ```sh
   cd src
   javac Chess.java
   ```
3. Run the project
   ```sh
   java Chess
   ```
